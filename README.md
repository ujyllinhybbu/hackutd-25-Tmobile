# PulseScore

## Inspiration
We wanted to build something that could make a real difference by combining live data and meaningful insights. Telecom users often face network issues that impact their experience, so we set out to analyze both real-time sentiment and network health to help companies respond faster and smarter.

## What it does
PulseScore is an AI-powered bot ticketing system that analyzes real-time customer sentiment during live conversations. It automatically flags tickets when frustration or negative sentiment is detected, allowing support teams to deliver faster, more personalized customer service. At the same time, PulseScore continuously tracks real-time network and performance data, helping companies identify patterns and make data-driven improvements to customer experience.

## How we built it
- **Frontend:** React + Tailwind CSS  
- **Backend:** Node.js (Express.js, Socket.IO, OpenAI)  
- **Database:** MongoDB  
- **Network Data:** Internet Health Report  

## Challenges we ran into
Some challenges we ran into included:  
- Displaying real-time data on graphs  
- Collecting and analyzing network data for potential issues such as alerts or overall status  

## Accomplishments that we're proud of
- Implementing automated flagging of tickets based on sentiment  
- Supporting real-time chat functionality  
- Collecting and visualizing real-time data efficiently
